# Security Policy

## Supported Versions

Support versions of ARM

| Version | Supported          |
|---------|--------------------|
| >=2.6   | :white_check_mark: |
| < 2.6   | :x:                |

## Reporting a Vulnerability

Please open an issue as with other requests.
Use [security] in your issue title this makes things easier, and it can be looked at sooner
